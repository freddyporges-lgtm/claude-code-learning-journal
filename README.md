# Claude Code Learning Journal

A personal log of discoveries, patterns, and growing fluency with Claude Code.

## Structure

```
entries/          # Daily session notes (YYYY-MM-DD.md)
synthesis/
  concepts.md     # Evergreen concept map — updated as understanding deepens
  weekly.md       # Rolling weekly synthesis
```

## How it works

At the end of (or during) a Claude Code session, trigger `/journal` in Claude Code.
Jeeves will reflect on what was explored, extract durable learnings, and append a new entry.
New concepts get folded into `synthesis/concepts.md`.

## Themes tracked

- MCP server configuration and authentication
- Claude Code architecture (tools, agents, skills, hooks)
- Workflow automation patterns
- Prompt engineering and agent design
